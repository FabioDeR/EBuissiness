À propos du projet

PtitGamelle est un projet E-Business complet développé avec des technologies modernes :

  • Frontend : React.js, TypeScript, Material-UI
  • Backend : Laravel (API RESTful)
  • Base de données : Relationnelle optimisée
  • Sécurité : JWT/Sanctum

Le projet comprend des interfaces pour les employés, restaurateurs et administrateurs, avec un système complet de gestion des plats, allergènes et disponibilités.
